## Description

[This project](https://github.com/sang-dang/NestAPI) is build on [Nest](https://github.com/nestjs/nest) framework.

## Clone this repository

```bash
$ git clone https://github.com/sang-dang/NestAPI
```

## Running the app

```bash
# development
$ npm start

# incremental rebuild (webpack)
$ npm run webpack

# production mode
$ npm run start:prod
```
